RndPhrase-XPI by jos

A Mozilla Firefox XPI for RndPhrase

RndPhrase XPI is a small Firefox extension to simplify the use of secure domain-specific passwords.

Dependencies

Mozilla Firefox 3 / Conkeror.

Install

Grab the source. Run pack.sh to pack the XPI file. Open the XPI file in Firefox to install the extension. You can now use autogenerated passwords by prefixing your regular passwords with "@".
